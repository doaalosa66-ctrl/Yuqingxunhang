## 🔗 API 接口对应关系

| 页面 | 调用接口 |
|------|---------|
| 01_搜索页面 | `POST /api/search` |
| 02_Forum运算态 | Socket.IO `search_progress` 事件 |
| 03_报告详情页 | `GET /api/report/result/<task_id>` |
| 04_历史报告列表 | `GET /api/report/history` |
| 05_系统配置页 | `GET/POST /api/config` |
| 06_控制台日志页 | `GET /api/output/<app>` |
| 07_操作指引页 | 纯前端，无 API 调用 |

## 🎨 样式说明

所有页面使用 **Space Theme（星空主题）** 设计系统：

### Design Tokens
- 背景色：`--space-void` (#0A0A0C)
- 毛玻璃：`--glass-bg` (rgba(255,255,255,0.05))
- 文字：`--text-silver` (#EBEBF5)
- 状态色：`--neon-green` / `--neon-blue` / `--neon-amber` / `--neon-red`
- 圆角：`--r-sm` (10px) / `--r-md` (16px) / `--r-lg` (22px)

### 动画
- 呼吸动画：`@keyframes breathe`
- 脉冲动画：`@keyframes pulse-travel`
- 打字机光标：`@keyframes blink`
