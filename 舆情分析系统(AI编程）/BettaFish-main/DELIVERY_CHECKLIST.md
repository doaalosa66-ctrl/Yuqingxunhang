## 🔄 回滚方案

### 回滚条件

如果出现以下情况，建议立即回滚：

1. 报告加载成功率 < 80%
2. 出现新的严重 Bug
3. 用户反馈负面评价 > 50%
4. 系统性能下降 > 20%

### 回滚步骤

```bash
# 1. 停止服务 (Ctrl+C)

# 2. 恢复备份
$backupFile = Get-ChildItem templates\index.html.backup.* | 
              Sort-Object LastWriteTime -Descending | 
              Select-Object -First 1
Copy-Item $backupFile.FullName templates\index.html -Force

# 3. 重启服务
python app.py
```
