# Mina AI舆情巡航系统 - 最小功能模块清单

> 基于当前系统架构拆分，每个模块为可独立开发、测试、交付的最小单元。

---

## 一、基础设施层

### 1.1 配置管理模块
- 功能：基于 Pydantic-Settings 的统一配置加载（.env / 环境变量 / 动态重载）
- 关键文件：`config.py`
- 配置项：Server、Database、LLM API Keys、Search API Keys、搜索限制参数

### 1.2 数据库连接模块
- 功能：异步数据库连接池管理，支持 PostgreSQL / MySQL 双驱动
- 关键文件：`InsightEngine/utils/db.py`
- 技术：SQLAlchemy 2.x async engine + asyncpg / aiomysql

### 1.3 Flask 主服务模块
- 功能：HTTP 服务启动、路由注册、SocketIO WebSocket 支持
- 关键文件：`app.py`
- 端口：默认 5000

### 1.4 Docker 部署模块
- 功能：容器化构建与编排
- 关键文件：`Dockerfile`、`docker-compose.yml`

---

## 二、搜索与数据采集层

### 2.1 Bocha 搜索工具
- 功能：Bocha AI 多模态搜索（文本/图片/视频，5 种搜索类型）
- 使用引擎：InsightEngine、MediaEngine
- 关键文件：`InsightEngine/tools/search.py`、`MediaEngine/tools/search.py`

### 2.2 Anspire 搜索工具
- 功能：Anspire AI 网页搜索
- 使用引擎：InsightEngine
- 关键文件：`InsightEngine/tools/search.py`

### 2.3 Tavily 新闻搜索工具
- 功能：Tavily 新闻搜索（6 种搜索工具）
- 使用引擎：QueryEngine
- 关键文件：`QueryEngine/tools/search.py`

### 2.4 MindSpider 爬虫 - 广域话题提取
- 功能：每日新闻抓取 + 话题自动提取
- 关键文件：`MindSpider/BroadTopicExtraction/`
- 子模块：`get_today_news.py`（新闻获取）、`topic_extractor.py`（话题分析）、`database_manager.py`（入库）

### 2.5 MindSpider 爬虫 - 深度情感爬取
- 功能：基于关键词的平台定向爬取
- 关键文件：`MindSpider/DeepSentimentCrawling/`
- 子模块：`keyword_manager.py`（关键词管理）、`platform_crawler.py`（平台爬虫）

### 2.6 MediaCrawler 社交媒体爬虫
- 功能：微博等社交媒体数据采集（Playwright 驱动）
- 关键文件：`MindSpider/MediaCrawler/`

---

## 三、AI Agent 引擎层

### 3.1 InsightEngine - 深度搜索 Agent
- LLM：Kimi-K2
- 功能：深度研究 + 反思迭代 + 聚类采样
- 处理流水线节点：
  - `FirstSearchNode` - 初始搜索
  - `ReflectionNode` - 反思迭代（最多 3 轮）
  - `FirstSummaryNode` - 内容摘要
  - `ReflectionSummaryNode` - 精炼摘要
  - `ReportFormattingNode` - 输出格式化
- 关键文件：`InsightEngine/agent.py`、`InsightEngine/nodes/`

### 3.2 InsightEngine - 关键词优化器
- LLM：Qwen-Plus
- 功能：SQL 查询关键词智能优化
- 关键文件：`InsightEngine/tools/keyword_optimizer.py`

### 3.3 InsightEngine - 情感分析工具
- 功能：多语言情感分析（22 种语言），含置信度评分
- 关键文件：`InsightEngine/tools/sentiment_analyzer.py`

### 3.4 InsightEngine - 聚类采样
- 功能：KMeans 聚类对搜索结果去重采样
- 关键文件：`InsightEngine/agent.py`（内嵌逻辑）

### 3.5 InsightEngine - 状态管理
- 功能：Search / Research 数据类，管理搜索历史、摘要、迭代状态
- 关键文件：`InsightEngine/state/state.py`

### 3.6 MediaEngine - 媒体分析 Agent
- LLM：Gemini-2.5-Pro
- 功能：多模态媒体内容搜索与分析
- 关键文件：`MediaEngine/agent.py`、`MediaEngine/nodes/`

### 3.7 QueryEngine - 查询处理 Agent
- LLM：DeepSeek-Chat
- 功能：新闻查询与时事研究
- 关键文件：`QueryEngine/agent.py`、`QueryEngine/nodes/`

### 3.8 ForumEngine - 论坛主持 Agent
- LLM：Qwen-Plus
- 功能：论坛讨论模拟与监控
- 关键文件：`ForumEngine/llm_host.py`、`ForumEngine/monitor.py`

### 3.9 LLM 客户端封装
- 功能：统一的 OpenAI SDK 兼容 LLM 调用封装
- 关键文件：`InsightEngine/llms/base.py`（各引擎复用同一模式）

### 3.10 Prompt 模板管理
- 功能：各引擎的提示词模板集中管理
- 关键文件：各引擎下 `prompts/prompts.py`

## 四、情感分析与 NLP 模型层

### 4.1 微博情感分析 - 微调模型
- 功能：基于 Transformer 微调的微博情感分类
- 关键文件：`SentimentAnalysisModel/WeiboSentiment_Finetuned/`

### 4.2 多语言情感分析模型
- 功能：支持 22 种语言的情感分析
- 关键文件：`SentimentAnalysisModel/WeiboMultilingualSentiment/`

### 4.3 机器学习情感分析
- 功能：XGBoost + Scikit-Learn 的轻量级情感分类
- 关键文件：`SentimentAnalysisModel/WeiboSentiment_MachineLearning/`

### 4.4 小型 Qwen 情感模型
- 功能：轻量级 Qwen 模型情感分析
- 关键文件：`SentimentAnalysisModel/WeiboSentiment_SmallQwen/`

### 4.5 BERT 话题检测模型
- 功能：基于 BERT 微调的话题自动检测
- 关键文件：`SentimentAnalysisModel/BertTopicDetection_Finetuned/`

## 五、报告生成层

### 5.1 ReportEngine - 报告编排 Agent
- LLM：Gemini-2.5-Pro
- 功能：协调多 Agent 结果，编排报告生成流程
- 关键文件：`ReportEngine/agent.py`

### 5.2 报告模板选择
- 功能：根据查询意图自动匹配 6 种预设模板
- 模板类型：
  - 企业品牌声誉分析报告
  - 市场竞争格局舆情分析
  - 日常/定期舆情监测
  - 特定政策或行业动态
  - 社会公共热点事件
  - 突发事件与危机公关
- 关键文件：`ReportEngine/nodes/TemplateSelectionNode`、`ReportEngine/report_template/`

### 5.3 文档布局设计
- 功能：根据模板设计文档结构与章节编排
- 关键文件：`ReportEngine/nodes/DocumentLayoutNode`

### 5.4 字数预算分配
- 功能：按章节重要性分配字数预算
- 关键文件：`ReportEngine/nodes/WordBudgetNode`

### 5.5 章节内容生成
- 功能：逐章节调用 LLM 生成报告正文
- 关键文件：`ReportEngine/nodes/ChapterGenerationNode`

### 5.6 章节存储与拼接
- 功能：章节持久化 + 文档组装
- 关键文件：`ReportEngine/core/chapter_storage.py`、`ReportEngine/core/stitcher.py`

### 5.7 JSON IR 中间表示
- 功能：报告的结构化中间表示（Intermediate Representation）
- 关键文件：`ReportEngine/ir/schema.py`

### 5.8 HTML 渲染器
- 功能：将 IR 渲染为 HTML 格式报告
- 关键文件：`ReportEngine/renderers/html_renderer.py`

### 5.9 PDF 渲染器
- 功能：HTML 转 PDF + 布局优化
- 关键文件：`ReportEngine/renderers/pdf_renderer.py`、`ReportEngine/renderers/pdf_layout_optimizer.py`

### 5.10 Markdown 渲染器
- 功能：将 IR 渲染为 Markdown 格式
- 关键文件：`ReportEngine/renderers/markdown_renderer.py`

### 5.11 图表 SVG 转换
- 功能：将图表数据转换为 SVG 嵌入报告
- 关键文件：`ReportEngine/renderers/chart_to_svg.py`

### 5.12 ReportEngine Flask 接口
- 功能：报告生成 HTTP/SSE API（任务提交、进度查询、结果下载）
- 关键文件：`ReportEngine/flask_interface.py`
- 端点：`/api/report/*`（共 13 个端点）

## 七、系统管理 API 层

### 7.1 引擎生命周期管理
- 功能：启动/停止/查询各引擎状态
- 端点：`POST /api/start/<app_name>`、`POST /api/stop/<app_name>`、`GET /api/output/<app_name>`

### 7.2 系统级控制
- 功能：全系统启动/关闭
- 端点：`POST /api/system/start`、`POST /api/system/shutdown`

### 7.3 配置管理 API
- 功能：运行时配置查询与更新
- 端点：`GET /api/config`、`POST /api/config`

### 7.4 论坛管理 API
- 功能：论坛启停与日志查询
- 端点：`GET /api/forum/start`、`GET /api/forum/stop`、`GET /api/forum/log`

### 7.5 WebSocket 实时通信
- 功能：SocketIO 实时状态推送
- 事件：`connect`、`request_status`、`disconnect`

### 7.6 系统状态 API
- 功能：系统健康检查
- 端点：`GET /api/status`
